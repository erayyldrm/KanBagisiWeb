# KanBagisiWeb
Kan bağışı süreci hakkıında bilgilendirmek ve kullanıcıların
yakınındaki bağış noktalarını görüntülemesini sağlamak amacıyla
geliştirilmiş web tabanlı bir uygulama.


## Proje Amacı

Bu proje, kan bağışının önemini vurgulamak, bağış süreci
hakkında kullanıcıları bilgilendirmek ve aktif bağış
noktalarının durumlarını listeleyerek bağış sürecini
kolaylaştırmak amacıyla geliştirilmiştir.

## Özellikler

- Kullanıcı dostu arayüz
- Kan bağışının faydaları hakkında bilgilendirme
- Yakındaki bağış noktalarını listeleme
- Bağış noktalarının durumlarını görüntüleme (aktif / pasif)
- Yönetici paneli üzerinden bağış noktası yönetimi



## Gelecek Hedefleri 
 Şuanda profil sayfam sadece navbar kısmında görüntü olarak var, 
 ilerleyen zamanlarda dinamik bir kullanıcı giriş sayfası yapmayı ve
 kullanıcı giriş yaptığında profil sayfasında kendi hakkında bilgiler
 getirilmesini amaçlanmaktayım ayrıca kan bağış noktaları içinde düzenleme
 ve geliştirme getirerek Şehir ve İlçe filtrelemesine göre bağış noktaları 
 getirmesi için düzenleme yapacağım.
